# あっきーの活動マインドマップ

琴平町地域おこし協力隊「あっきー」のAI・DAO活用による地域活性化活動の全体像を可視化したマインドマップです。

## 📂 ファイル構成

- **akki-activities.html** - マインドマップの表示用HTMLファイル
- **data/akki-activities.json** - マインドマップのデータファイル

## 🚀 使用方法

### ローカルで閲覧

1. このディレクトリで簡易サーバーを起動
   ```bash
   python -m http.server 8000
   ```

2. ブラウザで以下にアクセス
   ```
   http://localhost:8000/akki-activities.html
   ```

### ファイルをダブルクリックで開く

一部のブラウザでは、`akki-activities.html`を直接ダブルクリックしても表示可能です。
ただし、セキュリティ制限により一部機能が動作しない場合があります。

## 📝 データの更新

マインドマップのデータを更新する場合は、`data/akki-activities.json`ファイルを編集してください。

### データ構造

```json
{
  "centerNode": {
    "id": "root",
    "title": "タイトル",
    "type": "root",
    "description": "説明"
  },
  "nodes": [
    {
      "id": "ユニークID",
      "title": "ノードのタイトル",
      "type": "ノードタイプ",
      "description": "説明",
      "parentId": "親ノードのID"
    }
  ]
}
```

### ノードタイプ

- `root` - 中心テーマ
- `category` - カテゴリ
- `subcategory` - サブカテゴリ
- `success` - 成功事例
- `solution` - 対策・実施中
- `issue` - 課題・準備中
- `stats` - 実績
- `future` - 今後の展望
- `mission` - ミッション
